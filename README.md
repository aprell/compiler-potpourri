# Compiler Potpourri

A potpourri of programs, scripts, and tools for a compilers course covering:
- Three-address code
- Basic blocks
- Control flow graphs, dominators, and back edges of natural loops
- Data flow analysis
- Data dependences
- Loop transformations
- Parallel loops and tasks in OpenMP
- List scheduling
